# 📊 Insurance Analytics Dashboard | Power BI

## 📌 Project Overview
This Power BI dashboard provides comprehensive insights into insurance policies, premium collections, customer distribution, and policy performance. It enables stakeholders to monitor key business metrics and make data-driven decisions.

---

## 🎯 Objectives
- Analyze Total Premium Collection
- Track Maturity Amount and Coverage Amount
- Monitor Active Policies
- Compare Premium by Payment Frequency
- Analyze Policy Types and Customer Segments
- Evaluate Annual Premium Trends
- Support business decision-making through interactive dashboards

---

## 📈 Key KPIs
- Total Premium Paid
- Total Annual Premium
- Total Coverage Amount
- Total Maturity Amount
- Average Premium
- Number of Policies
- CAGR (%)
- Revenue Growth (%)
